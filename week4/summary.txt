This project demonstrates the implementation of an end-to-end data pipeline using Microsoft Azure services. A Resource Group and Storage Account were created, followed by a Blob Storage container where the Superstore.csv dataset was uploaded. An Azure Data Factory (ADF) instance was configured with a Linked Service to connect to the Storage Account. Source and destination datasets were created, and a pipeline was developed using the Get Metadata and Copy Data activities to validate the source file and copy it to a new location in Blob Storage. The pipeline was executed successfully and monitored within Azure Data Factory, providing practical experience with Azure cloud services, data integration, and ETL pipeline development.
